1. What is Semantic HTML? 
Semantic HTML is the use of HTML to reinforce the meaning of content on a web page, it helps search engines, screen readers, and other user devices determine the significance and context of web content.

2. What is HTML used for?
HTML stands for Hyper Text Markup Language and is used to format a web page. It informs the web browser what is included on the page.

3. What is an attribute and where do we put it? 
Attributes define additional properties of the element such as width and height of an image. Attributes are always specified in the start tag (or opening tag).

4. What is the h1 tag used for? How many times should I use it on a page?
H1 designates the main header on a page, it can only be used once per page.

5. Name two tags that have required attributes
Anchor <a> requires href <a href="">
Image <img> requires src <img src="">

6. What do we put in the head of our HTML document?
Information such as the page title , links to CSS file, and other metadata such as the author, and important keywords that describe the document.
 
7. What is an id?
The HTML id attribute is used to specify a unique id for an HTML element

8. What elements can I add an id to? 
Any HTML elements

9. How many times can I use the same id on a page? 
Only one element can have a specific id on a page.

10. What is a class?
A class is similar to an id in that it identifies certain elements, however it is not specific to one element. It can be used on as many elements as you would like.
 
11. What elements can I add a class to?
Any HTML element can have a class.
 
12. How many times can I use the same class on a page? 
As many times as you would like!

13. How do I get my link to open in a new tab?
Include the 'target' attribute after a link with a value of "_blank" 
<a href="https://www.lambdaschool.com" target="_blank">Lambda</a>

14. What is the alt attribute in the image tag used for? 
Alt is used to describe the picture in words in case the picture doesn't load or to let someone using a screenreader what the image is of.

15. How do I reference an id?
An ID selector in CSS is a name preceded by a hash character '#'

16. What is the difference between a section and a div
The <section> tag defines sections in a document, such as chapters, headers, footers, or any other sections of the document. The <div> tag defines a division or a section in an HTML document. The <div> tag is used to group block-elements to format them with CSS

17. What is CSS used for? 
Cascading Style Sheets

18. How to we select an element? Example - every h2 on the page
h2

19. What is the difference between a class and an id? - Give me an example of when I might use each one
A class may be used for multiple elements while an id can only be used on one element. A class is suitable for similar styling on different parts of a page while an id specifies a certain element, perhaps within a section or div that already has a class but requires different styling.

20. How do we select classes in CSS?
We use a full stop or "." before stating the class name.

21. How do we select a p element with a single class of “human””?
p.human

22. What is a parent child selector? When would this be useful? 
The CSS child selector has two selectors separated by a > symbol. The first selector indicates the parent element. The second selector indicates the child element CSS will style

23. How do you select all links within a div with the class of sidebar?
.sidebar

24. What is a pseudo selector?
A pseudo-class is a selector that selects elements that are in a specific state, such as they are the first element of their type, or they are being hovered over by the mouse pointer. They tend to act as if you had applied a class to some part of your document. :pseudo-class-name

25. What do we use the change the spacing between lines?
The 'line-height' property specifies the height of a line. 
div.a {
  line-height: normal (or 1.6, 80%);
}

26. What do we use to change the spacing between letters?
The 'letter-spacing' property specifies the spacing between characters. 
div.a {
   Letter-spacing: normal (or 2px);
}

27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?
The 'text-transform' property in CSS controls text case and capitalization. (uppercase, lowercase, capitalize, inherit)
.lowercase {
  text-transform: lowercase;
  }

28. How do I add a 1px border around my div that is dotted and black?

	border-width: 1px;
	border-color: black;
}

29. How do I select everything on the page?
The '*' selector allows you to select everything on the page.

Version control insures different versions will be compatible when they are merged together. It keeps code clean and working.

36. What is the command we would use to push our repo up to github? 
git push -u origin 'branch-name' will push all of your commits from your local machine up onto github

37. Walk me through Lambda's git flow. 
	1. Fork the repo you would like to work with on GitHub.
	2. Clone the forked repo from GitHub and copy the link.
	3. In CLI go to the directory you would like to save the clone using the 'cd' command.
	4. Clone the link using command 'git clone "link"'.
	5. Enter the clone using the 'cd' command.
	6. Create a new branch in the repo using 'git checkout -b "firstname-lastname"' command.
	7. Make changes to the repo.
	8. Commit changes on your branch using the 'git add .' command and add a comment of what you changed.
	9. Push changes back to GitHub repo using the 'git push -u origin branch-name' command.
	10. Submit a PR (pull request) on GitHub by clicking on the 'Branch:master' button and click on your branch new 	pull request.
	11. On the PR page, click on the dropdown button that says base repository and change it to your forked repo
	12. Do not merge but copy PR link and submit it on canvas.
